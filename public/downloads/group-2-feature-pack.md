# Group 2: Category Limits

## Mission

Stop users from overspending in one category.

Let users set spending limits for categories like food, transport, and data.

## Required Features

Add at least these five features to your own copy of the budget tracker.

1. Create a monthly spending limit for a category.
2. Show the remaining amount for each category.
3. Warn the user when a category is close to its limit.
4. List categories that have gone above their limit.
5. Reset category spending for a new month.

## Stretch Ideas

- Add default Nigerian student categories.
- Show the most used category.

## Beginner Instructions

1. First make sure the base budget tracker runs.
2. Add one feature at a time.
3. After each feature, run `python main.py`.
4. If the app breaks, undo only your last change and try again.
5. Prepare a short explanation of what each feature does.

## Presentation Tip

Do not only show code. Show the menu, add sample data, run your new feature, and explain the part of the code your group changed.
