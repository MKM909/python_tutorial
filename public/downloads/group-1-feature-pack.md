# Group 1: Savings Goals

## Mission

Help users save for something specific.

Add a savings-goal section to the budget tracker so users can track a target amount.

## Required Features

Add at least these five features to your own copy of the budget tracker.

1. Create a savings goal with a name and target amount.
2. Add money toward an existing savings goal.
3. Show the percentage completed for each goal.
4. Warn the user when an expense may affect their savings plan.
5. Display a goal summary that shows target, saved amount, and remaining amount.

## Stretch Ideas

- Add a goal deadline.
- Show goals sorted by closest to completion.

## Beginner Instructions

1. First make sure the base budget tracker runs.
2. Add one feature at a time.
3. After each feature, run `python main.py`.
4. If the app breaks, undo only your last change and try again.
5. Prepare a short explanation of what each feature does.

## Presentation Tip

Do not only show code. Show the menu, add sample data, run your new feature, and explain the part of the code your group changed.
