# Group 14: Mini Analytics

## Mission

Turn budget data into simple insights.

Add beginner-friendly analytics that explain the user budget in plain English.

## Required Features

Add at least these five features to your own copy of the budget tracker.

1. Show the top five highest expenses.
2. Calculate the average expense amount.
3. Calculate category percentages.
4. Print an ASCII chart of category spending.
5. Give a simple spending score with an explanation.

## Stretch Ideas

- Show best saving week.
- Suggest the category to reduce first.

## Beginner Instructions

1. First make sure the base budget tracker runs.
2. Add one feature at a time.
3. After each feature, run `python main.py`.
4. If the app breaks, undo only your last change and try again.
5. Prepare a short explanation of what each feature does.

## Presentation Tip

Do not only show code. Show the menu, add sample data, run your new feature, and explain the part of the code your group changed.
