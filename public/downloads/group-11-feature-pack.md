# Group 11: Monthly Report

## Mission

Prepare a clear report for one month.

Generate a month report students can show during presentation.

## Required Features

Add at least these five features to your own copy of the budget tracker.

1. Let the user choose a month to report on.
2. Calculate income, expenses, and balance for that month.
3. Find the biggest expense category for the month.
4. Print an ASCII bar chart for category totals.
5. Save the monthly report into a text file.

## Stretch Ideas

- Compare two months.
- Add best and worst spending days.

## Beginner Instructions

1. First make sure the base budget tracker runs.
2. Add one feature at a time.
3. After each feature, run `python main.py`.
4. If the app breaks, undo only your last change and try again.
5. Prepare a short explanation of what each feature does.

## Presentation Tip

Do not only show code. Show the menu, add sample data, run your new feature, and explain the part of the code your group changed.
