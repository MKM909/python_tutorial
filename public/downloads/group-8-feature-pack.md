# Group 8: Needs Wants Savings

## Mission

Classify spending using a simple money rule.

Teach users to split spending into needs, wants, and savings.

## Required Features

Add at least these five features to your own copy of the budget tracker.

1. Classify each expense as need, want, or savings.
2. Calculate the percentage spent in each class.
3. Compare spending with the 50-30-20 rule.
4. Warn when wants are too high compared with needs.
5. Print a simple text chart for the three classes.

## Stretch Ideas

- Let users choose their own rule.
- Suggest one area to reduce.

## Beginner Instructions

1. First make sure the base budget tracker runs.
2. Add one feature at a time.
3. After each feature, run `python main.py`.
4. If the app breaks, undo only your last change and try again.
5. Prepare a short explanation of what each feature does.

## Presentation Tip

Do not only show code. Show the menu, add sample data, run your new feature, and explain the part of the code your group changed.
