# Group 10: CSV Backup

## Mission

Move data in and out of the tracker.

Add CSV import and export so transaction data can be opened in spreadsheet apps.

## Required Features

Add at least these five features to your own copy of the budget tracker.

1. Export all transactions to a CSV file.
2. Import transactions from a CSV file.
3. Validate imported rows before saving them.
4. Use a backup filename that includes the current date.
5. Show how many records were imported or exported.

## Stretch Ideas

- Skip duplicate imported rows.
- Export only expenses or only income.

## Beginner Instructions

1. First make sure the base budget tracker runs.
2. Add one feature at a time.
3. After each feature, run `python main.py`.
4. If the app breaks, undo only your last change and try again.
5. Prepare a short explanation of what each feature does.

## Presentation Tip

Do not only show code. Show the menu, add sample data, run your new feature, and explain the part of the code your group changed.
