# Group 3: Time Summaries

## Mission

Help users understand spending over time.

Add daily and weekly summaries so users can see when they spend the most.

## Required Features

Add at least these five features to your own copy of the budget tracker.

1. Show total income and expenses for one selected day.
2. Show total income and expenses for one selected week.
3. Find the day with the highest spending.
4. Print a simple trend message such as spending increased or reduced.
5. Save the time summary into a text report file.

## Stretch Ideas

- Add monthly summary comparison.
- Show average spending per day.

## Beginner Instructions

1. First make sure the base budget tracker runs.
2. Add one feature at a time.
3. After each feature, run `python main.py`.
4. If the app breaks, undo only your last change and try again.
5. Prepare a short explanation of what each feature does.

## Presentation Tip

Do not only show code. Show the menu, add sample data, run your new feature, and explain the part of the code your group changed.
